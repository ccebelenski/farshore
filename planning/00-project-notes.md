# Project Notes

Inspired by Walter Bright's *Empire* (1977).

## Working from the planning docs

The planning docs are the project spec. Implementation proceeds from `planning/01-game-rules-spec.md` and the related design docs in this directory. If a spec is ambiguous, fix the spec — that's the path of least resistance.

## Naming

Type names and rule preset names are in descriptive terms: `BaselineAI`, `STANDARD`, `HeightFieldMapGenerator`.
